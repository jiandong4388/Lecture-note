# Lecture-note


这是`中国矿业大学的理学院Lecture Note的Latex 模板` ，如果你对这个模板，你只需做以下`三个` 步骤:  
1.`点击` 左上方的“clone and download”;  
2.`下载` 在本地，解压所有下载的压缩包;  
3.`打开` sample.tex的格式
